Decision Tree – Census Income Prediction

This project demonstrates a hands-on implementation of a Decision Tree model using the Census Income dataset. The goal is to preprocess the dataset, perform exploratory data analysis (EDA), and prepare the data for training a machine learning model.

Project Overview

In this project, we worked with a real-world dataset and applied essential data preprocessing and machine learning techniques.

The workflow includes:

Loading the dataset

Data exploration and understanding

Data cleaning and preprocessing

Handling categorical variables

Encoding features

Preparing the dataset for model training

Dataset

The dataset used is the Census Income dataset, which contains demographic and employment-related attributes used to predict income levels.

Some of the important features include:

Age

Workclass

Education

Occupation

Marital Status

Hours per week

Native Country

The final column is used as the target variable for prediction.

Exploratory Data Analysis (EDA)

EDA was performed to better understand the dataset structure.

Steps performed:

Checking dataset shape

Viewing column names

Inspecting data types

Identifying categorical variables

Handling missing or inconsistent values

Data Preprocessing

Several preprocessing steps were applied before model training:
Renaming the final column to target
Converting categorical variables into numerical format
Applying Label Encoding for object-type columns


Machine Learning Model

A Decision Tree model is used for classification. Decision Trees are powerful supervised learning algorithms that split the dataset based on feature conditions to make predictions.

Advantages of Decision Trees:

Easy to understand and interpret

Works well with both numerical and categorical data

Requires minimal data preprocessing
